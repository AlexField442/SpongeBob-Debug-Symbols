# SpongeBob-Debug-Symbols
These are debug symbols for various sixth-generation _SpongeBob_ games, ripped using Ghidra. They correspond to the following games:
* _SpongeBob SquarePants: Battle for Bikini Bottom_ (NTSC-U, GameCube)
* _The SpongeBob SquarePants Movie_ (NTSC-U, GameCube)
* _SpongeBob SquarePants: Creature from the Krusty Krab_ (NTSC-U, GameCube)
* _SpongeBob's Truth or Square_ (NTSC-U, Wii)
